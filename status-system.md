# 📊 ระบบ Status — SuPalArai (house-build app)

> เอกสารอธิบายระบบสถานะ/ธง/สิทธิ์ ในการ์ดรายการ (collapse bar) ของแอป
> อัปเดต: 2026-06-07 · ไฟล์แอป: `app/index.html`

---

## 1) สถานะหลัก (5 สถานะ) — เลือกได้ทีละอัน
กำหนดใน `STATUSES` + `STATUS_META` (สี/สัญลักษณ์)

| สถานะ | วงกลม | สี/สัญลักษณ์ | คลาส CSS |
|---|---|---|---|
| รอเริ่ม | ⚪ | ขาว ขอบเทา | `st-wait` |
| กำลังทำ | 🟢 | เขียวทึบ | `st-doing` |
| เสร็จ | ✅ | เขียว + เครื่องหมาย ✓ | `st-done` |
| ติดปัญหา | 🟠 | ส้ม | `st-issue` |
| ยกเลิก | ⊗ | ขาว + กากบาท ✕ แดง | `st-cancel` |

- แสดงเป็นวงกลมที่หัวบาร์ (ย่อ) + แถวปุ่มเลือกตอนกาง
- หน้าภาพรวม (dashboard) นับ "เสร็จ" เป็นความคืบหน้า
- รายการสถานะ "เสร็จ" จะถูกเรียงไปท้ายลิสต์อัตโนมัติ

## 2) ธงแจ้งเตือน 🔔 "ต้องการข้อมูล/ปรึกษาเพิ่ม" (`needHelp`)
- เป็น **ธงแยก** จากสถานะหลัก (เปิด/ปิดได้อิสระ)
- เปิดธง → **วงกลมสถานะหลักกระพริบ** ตามสีตัวเอง
  - กำลังทำ → เขียวกระพริบ
  - ติดปัญหา → ส้มกระพริบ
  - **รอเริ่ม → กระพริบ ขาว-ดำ** (สลับสี เพราะขาวจางบนพื้นครีมจะมองไม่เห็น)
- **ยกเว้น "เสร็จ" และ "ยกเลิก"** — กดธงไม่ได้ + ไม่กระพริบ (ถ้าเปลี่ยนสถานะเป็น 2 อันนี้ ธงจะปิดเองอัตโนมัติ)
- ฟังก์ชันเช็ค: `canBlink(status)` = สถานะที่ไม่ใช่ เสร็จ/ยกเลิก

## 3) สิทธิ์การแก้ไข
- **ผู้รับผิดชอบ (owner)**: non-admin เลือกได้แค่ตัวเอง/เว้นว่าง · admin เลือกได้ทุกคน
- **รายละเอียด (detail)**: แก้ได้เฉพาะ **ผู้รับผิดชอบ** + **admin** (ถ้ายังไม่มีผู้รับผิดชอบ → ผู้สร้างแก้ได้) · กรอบ **แดง**
- **โน้ต (note)**: **ทุกคนเพิ่มได้** · กรอบ **น้ำเงิน** · ติดชื่อผู้เขียน + วันที่ (ตัวจางเอียง)
  - แก้ได้เฉพาะโน้ตที่ตัวเองเขียน (admin แก้ได้ทุกอัน) → เมื่อแก้จะติด "(แก้ <วันที่>)" หลังชื่อ

## 4) ฟิลด์ในการ์ด 1 รายการ
`{ title, sub, creator, owner, created(วันที่เริ่ม), due(วันที่คาดเสร็จ), status, needHelp, detail, notes[], images[] }`
- วันที่แสดงแบบไทยสั้นในบาร์: `5มค69 - 7กพ69` (วันเริ่ม - วันคาดเสร็จ · เดือนย่อ + พ.ศ. 2 หลัก)

## 5) อ้างอิงโค้ด (app/index.html)
- `STATUSES`, `STATUS_META`, `NO_BLINK`, `canBlink()`, `statusDot()`
- `statusSym()` — สัญลักษณ์ ✓ (เสร็จ) / ✕ (ยกเลิก)
- CSS: `.sdot.st-*`, `.sdot.blink` (จางทั่วไป), `.sdot.st-wait.blink` (ขาว-ดำ)
- `canEditDetail(it)`, `isAdmin()`
- โน้ต: `addNote()`, `startEditNote()`, `saveEditNote()`
- รูป: `attachImage()`, `delImage()`
